# Alpine.js 素振り

Alpine.js の基本機能を試すためのサンプル集です。

https://masutaka.github.io/trial-alpinejs/

## ローカルで確認する場合

```bash
npx http-server -p 8000
```

http://localhost:8000 にアクセス

## 含まれるサンプル

- カウンター
- 表示/非表示の切り替え
- 入力フォームとバインディング
- ループ
- x-show による条件分岐
- x-if による条件分岐
