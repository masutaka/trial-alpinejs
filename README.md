# Alpine.js 素振り

Alpine.js の基本機能を試すためのローカル環境です。

## セットアップ

特別なインストールは不要です。ローカルサーバーを起動するだけで使えます。

## 使い方

1. ローカルサーバーを起動

```bash
npx http-server -p 8000
```

2. ブラウザで開く

http://localhost:8000 にアクセス

## 含まれるサンプル

- カウンター
- 表示/非表示の切り替え
- 入力フォームとバインディング
- ループ
- x-show による条件分岐
- x-if による条件分岐
